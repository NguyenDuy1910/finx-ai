# FinX UI

Text2SQL AI chatbot frontend built with Next.js, TypeScript, and Tailwind CSS.

## Setup

```bash
npm install
cp .env.local.example .env.local  # or create .env.local with BACKEND_URL=http://localhost:8000
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Backend

Requires the `finx-agentic` FastAPI backend running on `http://localhost:8000`.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS v4
- Lucide React icons
